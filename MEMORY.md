## 重要经验

### PR 审核最佳实践
1. 使用标准格式：评分/优势亮点/改进建议/建议行动/市场潜力评估/结论
2. 平衡鼓励与建设性批评
3. 给出具体的可执行建议
4. 注意项目的独特价值点

### 常见问题
- 大型PR应建议拆分
- 技术风险需要缓解措施
- 商业模式需要验证路径
- 数据隐私合规需要关注
