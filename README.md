## 🌟 这是什么？

**AI Ideas** 是一个开源的 AI 创意收集与孵化仓库，致力于记录、讨论和实现各种有价值的 AI 应用想法。

不同于简单的灵感列表，这里的每个创意都包含：
- 📋 **完整的产品设计** — 功能、用户场景、差异化定位
- 🔧 **技术实现方案** — 架构、API、成本估算
- 💰 **商业模式分析** — 变现路径、竞品对比、单位经济
- 📊 **可落地的 roadmap** — 分阶段实施计划

## 🎯 如何参与

### 💡 提交新创意

1. **有初步想法** → 提交到 [`proposals/`](./proposals) 目录
2. **想法较成熟** → 直接提交到 [`ideas/`](./ideas) 目录
3. **有完整设计** → 创建 PR 到 [`pr/`](./pr) 目录

### 📝 创意文档模板

详见 [`TEMPLATE.md`](./TEMPLATE.md)，每个创意建议包含：

```markdown
# 创意名称

> 一句话卖点

## 概述
## 功能设计
## 技术方案
## 实现步骤
## 资源需求
## 变现模式
## 竞品分析
## 风险与缓解
```

### 🔍 参与讨论

- 对现有创意有疑问？→ [打开 Issue](../../issues/new)
- 想认领实现某个创意？→ 在创意文档下留言
- 有改进建议？→ 直接提交 PR
