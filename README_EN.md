## 🌟 What is this?

**AI Ideas** is an open-source incubator for AI product ideas — documenting, discussing, and realizing valuable AI applications.

Unlike simple idea lists, each idea here includes:
- 📋 **Complete Product Design** — Features, user scenarios, differentiation
- 🔧 **Technical Implementation** — Architecture, APIs, cost estimation
- 💰 **Business Model Analysis** — Monetization, competitive analysis, unit economics
- 📊 **Actionable Roadmap** — Phased implementation plans

## 🎯 How to Contribute

### 💡 Submit a New Idea

1. **Early concept** → Submit to [`proposals/`](./proposals)
2. **Well-formed idea** → Submit directly to [`ideas/`](./ideas)
3. **Complete design** → Create a PR to [`pr/`](./pr)

### 📝 Idea Template

See [`TEMPLATE.md`](./TEMPLATE.md). Each idea should include:

```markdown
# Idea Name

> One-liner pitch

## Overview
## Feature Design
## Technical Solution
## Implementation Steps
## Resource Requirements
## Monetization
## Competitive Analysis
## Risks & Mitigation
```

### 🔍 Join the Discussion

- Questions? → [Open an Issue](../../issues/new)
- Want to build an idea? → Comment on the idea document
- Have improvements? → Submit a PR
