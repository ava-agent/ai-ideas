# AI 照片故事生成器

> 把手机里沉睡的照片，变成动人的故事

## 核心功能

### 1. 智能分析
- **时间维度** - 识别季节、节日、特殊日期
- **地点维度** - 旅行路线、常去的地方
- **人物识别** - 家人、朋友、宠物
- **场景理解** - 生日派对、户外徒步、美食记录

### 2. 故事生成
- 自动生成时间线
- 精选 9-12 张代表性照片
- 配上温馨的文字描述
- 支持多种风格（温馨/幽默/文艺）

### 3. 一键导出
- 精美图文排版
- 短视频生成
- 直接分享到社交媒体

## 技术方案

### 架构
```
照片输入 → 元数据提取 → 多模态分析 → 故事生成 → 导出
   ↓            ↓            ↓           ↓        ↓
 EXIF/HEIC   时间/地点    GPT-4V/LLaVA  LLM    Canva/FFmpeg
```

### 核心组件

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 元数据提取 | exiftool | 提取时间、GPS、设备信息 |
| 图像理解 | GPT-4V / LLaVA | 场景、人物、情感分析 |
| 故事生成 | GLM-5 / Claude | 时间线 + 文案生成 |
| 排版渲染 | Canva API / HTML | 精美图文模板 |
| 视频生成 | FFmpeg + MoviePy | 照片轮播 + 转场 |

### 隐私方案

**优先级：端侧处理**
1. 本地提取 EXIF（无上传）
2. 本地 LLaVA 模型（可选）
3. 云端处理需用户明确授权
4. 照片不存储，处理完即删

## 竞品分析

| 产品 | 优势 | 劣势 |
|------|------|------|
| Google Photos 回忆 | 集成度高 | 排版固定，分享弱 |
| iOS 回忆 | 体验流畅 | 生态封闭 |
| Instagram 精选 | 社交强 | 无故事线 |

**差异化**：精美排版 + 故事性 + 一键分享
