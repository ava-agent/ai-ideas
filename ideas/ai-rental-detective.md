# AI 租房看房侦探

> **一句话卖点**：租房者拍照合同，AI 3 秒标注风险条款，避免被坑

## 概述

基于 Issue #115 创建的详细设计文档。

中国租房市场痛点极强——中介话术真假难辨，看房时间短看不出潜在问题，合同条款复杂容易踩坑。**AI 租房看房侦探**通过多模态 AI 技术，为租房小白提供「看房全流程 AI 助手」，10 分钟内识别房源问题、分析周边配套、扫描合同风险。

**核心价值：** 标准化诊断 + 个性化建议，做「看房必备」的品类第一。

## 功能设计

### 核心功能

1. **房源拍照诊断**
   - 渗水痕迹识别：AI 标注墙面、窗边渗水区域
   - 采光分析：识别窗户朝向，评估全天采光情况
   - 插座数量检测：识别插座位置并计数，判断是否满足需求
   - 基础设施检查：空调、热水器、燃气灶等设备状态

2. **周边智能分析**
   - 真实通勤时间：基于定位，计算到公司/学校的实际步行/骑行时间
   - 周边配套聚合：餐饮、医院、学校、超市等 POI 数据
   - 治安情况评估：公开数据 + 社区评价情感分析

3. **租金价格对比**
   - 同区域同户型租金数据对比
   - 判断是否溢价，给出合理价格区间
   - 相似房源匹配算法（面积、朝向、楼层、地铁距离）

4. **合同风险扫描**
   - 拍照/上传合同，AI 3 秒标注风险条款
   - 识别隐藏费用、不合理押金条款、违约责任不对等
   - 生成风险评分和建议

5. **个性化看房清单**
   - 根据用户需求（养猫、做饭、居家办公、通勤）生成检查清单
   - 包含必查项和加分项
   - 支持自定义添加关注点

### 用户体验亮点

- **10 分钟快速诊断**：从拍照到完整报告，全流程不超过 10 分钟
- **可视化报告**：图文并茂，问题点高亮标注
- **分享功能**：可生成报告链接，发给室友/家人共同决策
- **历史记录**：保存已看房源报告，方便对比选择

## 技术方案

### 数据层

**多模态数据采集：**
- 房间照片/视频（用户上传）
- 合同照片/PDF（OCR 处理）
- GPS 定位数据
- 周边 POI 数据（高德/百度 API）
- 租金数据（贝壳/链家公开数据 + 用户众包）

**数据存储：**
- 图片存储：阿里云 OSS / 腾讯云 COS
- 结构化数据：PostgreSQL + PostGIS（地理空间查询）
- 缓存层：Redis（周边数据缓存）

### 逻辑层

**图像识别模块：**
- **渗水识别**：SAM（Segment Anything Model）定位异常区域 + 分类模型判断
- **采光分析**：
  - 窗户朝向识别（基于太阳位置的几何计算）
  - 室内亮度评估（HSV 色彩空间分析）
- **插座检测**：YOLOv8 目标检测
- **设备识别**：预训练分类模型（空调、热水器等）

**合同分析模块：**
- **OCR 引擎**：百度 OCR / 腾讯 OCR（中文优化）
- **条款识别**：GPT-4V + 规则引擎混合方案
- **风险标注**：LLM 分析 + 法律知识库（RAG 架构）

**周边分析模块：**
- **通勤计算**：高德/百度地图 API + 实时路况数据
- **配套聚合**：POI 数据聚合（餐饮、医院、学校等）
- **治安评估**：公开数据 + 社区评价情感分析（NLP）

**租金对比模块：**
- **数据源**：贝壳/链家公开数据 + 用户众包数据
- **相似度算法**：余弦相似度（面积、朝向、楼层、地铁距离等特征）
- **价格预测**：LightGBM 回归模型

### 展示层

**前端架构：**
- 小程序（微信/支付宝）为主，H5 为辅
- 地图组件：高德地图 SDK
- 图像标注：Canvas 绘制 + 可视化库

**交互设计：**
- 拍照引导：AI 实时提示「请拍摄窗户」「请拍摄合同首页」
- 问题标注：点击高亮区域查看详细说明
- 报告导出：支持 PDF / 图片分享

## 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| **OCR 准确率低** | 合同识别效果差 | 采用 GPT-4V + 人工模板混合方案，初期聚焦常见合同格式 |
| **数据合规风险** | 爬取租房平台数据违规 | 使用公开 API + 用户众包数据，避免直接爬取 |
| **GPS 定位不准** | 周边分析偏差 | 提供手动标记位置功能，拍摄小区门口辅助定位 |
| **竞品威胁** | 链家/贝壳加 AI 功能 | 快速占领心智，积累「房源缺陷数据库」形成壁垒 |
| **用户隐私担忧** | 不愿上传合同照片 | 本地 OCR 选项 + 明确的隐私政策 + 数据加密 |
| **冷启动数据不足** | 租金对比不准 | 初期聚焦一线城市热门区域，逐步扩展数据覆盖 |

## 竞品对比

| 竞品 | 优势 | 劣势 | 我们的差异化 |
