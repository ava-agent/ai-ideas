# AI 跑步陪练 - 从跑500米就喘到轻松完成首个5公里

> **一句话卖点**：跑步新手在每次晨跑时，用 AI 陪练获得实时语音鼓励和配速指导，从跑500米就喘到3周后轻松完成首个5公里。

## 概述

基于 Issue #174 创建的详细设计文档。

这是一个面向跑步新手的 AI 陪练产品，通过实时语音陪伴、动态配速指导和情绪激励，帮助零基础用户跨越「从0到5公里」的心理和生理障碍。

**核心价值**：不是冰冷的配速数据，而是有温度的「我陪着你跑」。

## 功能设计

### 核心功能 1：实时语音陪跑
- 耳机实时鼓励：「还有200米，保持住！」、「调整呼吸，吸气-吸气-呼」
- 延迟 < 1秒，确保不影响跑步节奏
- 支持多种语音风格（温和型/激励型/教练型）

### 核心功能 2：动态配速指导
- 根据实时心率自动调整配速建议
- 基于心率区间训练（最大心率的 60-70% 为燃脂区）
- 动态提醒：「慢一点，保持这个节奏」、「可以稍微加速了」

### 核心功能 3：跑姿监测（Phase 2）
- 通过手机摄像头进行姿态估计
- 检测步频、步幅、身体前倾角度
- 实时提醒：「肩膀放松，核心收紧」

### 核心功能 4：情绪加油站
- 跑不动时的正向反馈：「你已经跑完2公里了！比上周进步30%」
- 疲劳检测 + 个性化鼓励语
- 社交压力转化：「朋友圈打卡模板已生成」

## 技术方案

### 数据层
- **用户数据**：本地 SQLite + 云端同步
- **训练记录**：时序数据库（TimescaleDB）用于分析配速趋势
- **语音缓存**：预生成常用语音包，减少实时生成延迟

### 逻辑层
- **语音陪跑引擎**：
  - TTS（语音合成）：Edge TTS / Azure Speech
  - 延迟控制：< 1秒，预生成 + 实时生成混合策略
  - 离线支持：核心语音包本地化

- **配速指导算法**：
  - 输入：实时心率 + 用户体感反馈 + 历史数据
  - 算法：规则引擎 + 轻量 ML 模型
  - 输出：配速调整建议

- **跑姿检测**（Phase 2）：
  - 技术：MediaPipe（姿态估计）+ TensorFlow Lite
  - 挑战：跑步时手机固定方式影响准确率
  - 降级方案：跑后分析 + 视频回放标注

### 展示层
- **移动端**：React Native（iOS + Android）
- **穿戴设备集成**：Apple Watch / 华为手环 / 小米手环
- **语音交互**：蓝牙耳机 + 手机扬声器

## 实现步骤

### Phase 1：MVP（2周）
1. 基础语音陪跑（距离里程碑播报）
2. 手动输入心率/体感的配速建议
3. C25K 训练计划（固定方案）
4. 简单的完成记录和进度追踪

**验证目标**：用户是否喜欢「有人陪着跑」的感觉？

### Phase 2：智能配速（2周）
1. 接入 Apple Watch / 手机心率数据
2. 基于心率的动态配速调整
3. AI 生成个性化鼓励语（接入 LLM）
4. 情绪加油站（疲劳检测 + 正向反馈）

**验证目标**：智能配速是否真的提升了训练效果？

### Phase 3：跑姿监测（2周）
1. 手机摄像头姿态估计（MediaPipe）
2. 实时跑姿提醒
3. 跑后姿势分析报告
4. 纠正训练视频推荐

**验证目标**：跑姿监测的准确率和用户接受度

### Phase 4：社区与变现（2周）
1. 虚拟跑团（和 AI 或其他用户一起跑）
2. 挑战赛（完成 5K、10K、半马解锁徽章）
3. 付费墙设计（Freemium 模式）
4. 社交分享功能

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| 实时语音延迟（户外网络不稳定） | 预生成常用语音包 + 本地缓存；网络差时切换为预设语音包，不等云端合成 |
| 穿戴设备 API 碎片化（Apple/华为/小米/佳明各有协议） | Phase 1 先支持 Apple Watch + 手机扬声器，验证核心体验；逐步扩展其他设备 |
| 跑姿检测准确率受环境影响（光线、角度） | 初期用手机摄像头做跑后分析，实时检测需额外穿戴设备；提供「手动输入」作为备选 |
| 用户学会后流失 | 延伸到「10K、半马进阶计划」；引入社区和挑战赛增加粘性 |
| 隐私敏感（位置、心率数据） | 本地优先架构 + 端到端加密 + 用户完全控制数据 |

## 差异化竞争优势

| 维度 | 现有方案（Keep/nrc） | 本产品 |
|------|---------------------|--------|
| 陪伴感 | 跑完看数据，自己分析 | 跑步时实时语音反馈 |
| 训练计划 | 静态的 PDF/视频 | 根据实时状态动态调整 |
| 情绪价值 | 只有数据，没有情感 | 「你可以的！」情绪激励 |
| 孤独感 | 一个人跑很孤独 | AI 教练全程陪伴 |
| 新手友好度 | 需要一定基础 | 从零基础到5公里的完整路径 |
