## 📋 项目概述

### 核心问题

心理咨询师目前主要依靠主观经验和客户描述进行情绪判断，存在以下痛点：
- 判断依赖个人经验，准确率参差不齐
- 客户描述不完整或不准确，影响诊断质量
- 缺乏客观数据支撑，治疗效果难以量化
- 咨询过程中情绪变化缺乏实时监测

### 解决方案

AI情绪图谱分析师：一套完整的多模态情绪分析系统，通过语音情感分析、面部表情识别、文本情感分析构建动态情绪图谱，为心理咨询师提供数据化决策支持。

## 🏗️ 核心架构

### 多模态数据采集

```
┌─────────────────────────────────────────────┐
│          多模态数据采集系统                     │
├─────────────────────────────────────────────┤
│  语音分析  │  表情识别  │  文本分析  │  数据融合  │
└─────────────────────────────────────────────┘
```

**语音情感分析**
- 使用OpenSmile提取声学特征
- Wav2Vec2进行语音情感分类
- 支持普通话、粤语、英语
- 实时情感状态跟踪

**面部表情识别**
- 基于FER+的面部表情识别
- 微表情捕捉和分析（准确率 60-75%）
- 眼神接触和专注度评估
- 面部动作单元(AU)分析

**文本情感分析**
- RoBERTa中文情感模型
- 语义理解和情感极性分析
- 关键情绪词提取和分类
- 上下文相关情感推断

**数据融合引擎**
- 多模态时间序列对齐
- 情绪图谱动态构建
- 异常情绪模式识别
- 风险预警机制

### 情绪图谱构建

```
输入数据 → 特征提取 → 时间对齐 → 图谱构建 → 模式识别
```

**图谱结构**
- **节点**: 情绪状态、事件触发、生理反应
- **边**: 关联强度、时间关系、因果联系
- **权重**: 情绪强度、可信度、重要性
- **动态更新**: 实时添加新节点和关系

**分析算法**
- 社区发现算法识别情绪聚类
- 时间序列分析发现情绪模式
- 异常检测识别风险信号
- 推荐引擎生成干预建议

- **情绪模式匹配**: 与历史案例对比，相似度分析

### 4. 隐私保护系统

- **端到端加密**: 所有数据传输和存储加密
- **本地化处理**: 敏感信息本地分析，不上传云端
