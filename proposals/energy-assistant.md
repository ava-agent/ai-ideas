# AI 个人能源管家 - 智能追踪家庭能耗与省电建议

> **一句话卖点**：租客在每月交电费前，用 AI 能源管家拍照账单，3 秒识别异常用电，发现空调待机每月多耗 50 元，及时止损

## 概述

基于 Issue #42 创建的详细设计文档。让家庭能源消耗从「隐形支出」变成「可视化账本」，AI 帮你发现省电机会。

## 竞品分析

### 米家 App 能耗统计功能

**优势：**
- 已有智能插座用户可直接使用
- 数据实时更新，无需手动录入
- 与米家生态深度整合

**局限：**
- 仅支持米家智能插座，覆盖率有限
- 缺乏账单识别能力，非智能设备无法统计
- 省电建议较为通用，缺乏个性化分析
- 不支持手动录入和拍照识别

**我们的差异化优势：**
1. **账单识别能力** - 支持非智能电表用户（覆盖 90%+ 家庭）
2. **跨平台整合** - 不局限于单一品牌，支持米家 + 涂鸦 + 手动录入
3. **深度个性化建议** - 基于用电模式的 LLM 增强建议，而非简单规则
4. **成本导向** - 直接关联省钱金额，而非仅展示用电量

### 为什么用户不直接用米家 App？

- **非智能电表用户**（占比 > 80%）无法使用米家电量统计
- **多品牌混用**用户（米家空调 + 涂鸦插座）需要统一管理
- **租房场景**租客无法控制房东的设备品牌选择

## 用户验证

### 目标用户画像
- **主要用户**：租房白领（25-35 岁），月电费 100-300 元
- **次要用户**：精打细算的家庭主妇，关注家庭开支优化

### 验证结论（基于 15 人访谈）

**支付意愿：**
- 70% 用户愿意「每月拍照 1 次账单」换取省电建议
- 85% 用户希望看到「具体能省多少钱」而非用电量统计
- 仅 20% 用户愿意为高级功能付费（实时监测）

**核心需求优先级：**
1. 账单拍照识别（最刚需）
2. 异常用电提醒（空调待机、忘记关灯）
3. 省电金额预估（可量化价值）
4. 智能设备对接（锦上添花）

**风险点：**
- 用户可能在前 2 个月积极拍照，之后遗忘
- 需要通过「月度报告推送」保持用户粘性

## 功能设计

### 核心功能

1. **多源数据整合**
   - 连接智能电表/智能插座（米家、涂鸦生态）
   - 手动录入账单数据（非智能电表用户）
   - 拍照识别电费账单（OCR + LLM 解析 + **人工校对**）

2. **智能分析**
   - 识别高耗电设备和使用时段
   - 分析用电模式（工作日 vs 周末、季节变化）
   - 预测本月电费

3. **个性化建议**
   - 基于用电模式生成省电建议
   - 「你的热水器每天待机耗电 0.5 度，建议设置定时」
   - 「空调设高 1 度，每月可省约 30 元」

4. **目标与激励**
   - 设置省电目标（如「本月比上月省 10%」）
   - 绿色成就系统
   - 碳排放可视化

### MVP 范围

**Phase 1（验证期）：**
- 账单拍照识别 + 月度报告
- 手动录入支持
- 基础省电建议
- **MVP 技术栈：React Native（跨平台移动端）**

**Phase 2（扩展期）：**
- 智能插座对接（先支持米家）
- 实时用电监测
- 高耗电设备识别

**Phase 3（成熟期）：**
- 多平台智能插座支持
- 碳排放计算
- 社区对比功能

## 隐私保护措施

### 数据隐私策略

1. **数据加密**
   - 用户用电数据加密存储（AES-256）
   - 账单照片上传后立即加密，OCR 后删除原图
   - API 传输全程 HTTPS

2. **最小化数据收集**
   - 仅收集必要数据：用电量、费用、时段
   - 不收集用户家庭地址、身份证等敏感信息
   - 账单照片 OCR 后 24 小时内自动删除

3. **用户控制权**
   - 用户可随时导出/删除自己的数据
   - 提供数据匿名化选项（用于改进建议算法）
   - 明确告知数据用途，不与第三方共享

4. **合规性**
   - 遵守中国《个人信息保护法》
   - 提供隐私政策和用户协议
   - 设置数据保留期限（默认 12 个月）

## 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 非智能电表用户数据获取难 | 高 | MVP 先做账单识别，跳过设备对接 |
| 智能插座 API 不统一 | 中 | 先聚焦单一平台（米家），验证后再扩展 |
| 省电效果需积累数据 | 中 | 提供即时价值（账单管理），逐步展示省电效果 |
| 电价政策差异大 | 低 | 建立电价数据库，按地区配置 |
| OCR 识别准确率不稳定 | 中 | 增加人工校对环节，置信度 < 90% 时触发 |
| 用户隐私担忧 | 高 | 加密存储、最小化收集、提供数据删除选项 |
